# لوحة تحكم الآيلتس الذكية 🎯

لوحة تحكم تفاعلية لخطة تحضير الآيلتس في 6 أسابيع — صفحة واحدة (SPA) بدون أي حاجة لسيرفر أو Backend.

## 📁 محتوى المشروع

```
ielts-dashboard/
├── index.html   ← الصفحة الرئيسية (الهيكل)
├── style.css    ← التنسيقات المخصصة (الأنيميشن، الـ checkbox، إلخ)
├── script.js    ← المنطق والبيانات (المصادر، الأسابيع، التخزين المحلي)
└── README.md    ← هذا الملف
```

## 🚀 طريقة رفع المشروع على GitHub

### 1. إنشاء مستودع جديد
- اذهب إلى [github.com/new](https://github.com/new)
- اختر اسم للمستودع، مثلاً: `ielts-dashboard`
- اجعله **Public** (لتتمكن من نشره مجاناً عبر GitHub Pages)
- لا تُفعّل "Add a README" لأنك سترفع الملفات يدوياً

### 2. رفع الملفات (طريقتان)

**أ) عبر المتصفح (أسهل):**
1. افتح المستودع الجديد
2. اضغط **"uploading an existing file"**
3. اسحب الملفات الثلاثة (`index.html`, `style.css`, `script.js`) معاً
4. اضغط **Commit changes**

**ب) عبر Git في الطرفية:**
```bash
git clone https://github.com/USERNAME/ielts-dashboard.git
cd ielts-dashboard
# ضع الملفات الثلاثة هنا
git add .
git commit -m "إضافة لوحة تحكم الآيلتس"
git push origin main
```

### 3. تفعيل GitHub Pages (لنشر الموقع كرابط مباشر)
1. من المستودع، اذهب إلى **Settings** → **Pages**
2. تحت **Source**، اختر الفرع `main` والمجلد `/ (root)`
3. اضغط **Save**
4. بعد دقيقة أو دقيقتين، سيظهر رابط الموقع بالشكل:
   ```
   https://USERNAME.github.io/ielts-dashboard/
   ```

## 🖥️ التشغيل المحلي (بدون رفع)
فقط افتح ملف `index.html` مباشرة في أي متصفح — لا حاجة لأي تثبيت.

## ⚙️ التقنيات المستخدمة
- **HTML5** — الهيكل
- **Tailwind CSS** (عبر CDN) — التنسيق
- **JavaScript (Vanilla)** — التفاعل والمنطق
- **localStorage** — حفظ تقدّم المهام محلياً في متصفح المستخدم (لا يوجد قاعدة بيانات أو سيرفر)

## 📝 ملاحظة على البيانات
كل بيانات المصادر والخطة الأسبوعية موجودة داخل `script.js` في متغيرين:
- `RESOURCES` — قنوات ومواقع التحضير
- `WEEKS` — تفاصيل المهام لكل أسبوع

لتعديل أي رابط أو مهمة، عدّل القيم في هذين المتغيرين مباشرة.
